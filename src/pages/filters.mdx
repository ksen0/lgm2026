  - snippet: strands
    description: <h1>Writing Shaders with p5.strands</h1><p>In p5.js v2, shaders can be written using JavaScript which is then transpiled to GLSL.</p><p>A p5.strands shader has to be defined first (e.g., <code>buildFilterShader(...)</code>) and then used (e.g., <code>filter(...)</code>). In the definition, different hooks can be used to access and update data. For example, getting the current color at a pizel and changing it.</p>
